Welcome to AWS Redshift Data Warehouse's documentation!
=======================================================

**aws-dwh** is a Python project for extract data from JSON files stored
in S3 buckets and load into Redshift Data Warehose tables.

.. note::

   This project is part of the `Udacity's Data Engineeering Nanodegree Program <https://www.udacity.com/course/data-engineer-nanodegree--nd027>`_.
